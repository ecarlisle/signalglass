# Spec 011: CLI

## Status

Implemented (analyze), Draft (ingress/traces)

## Purpose

Define the command-line interface for Signalglass.

## Scope

- Offline analysis command.
- Live ingress command.
- Trace management commands.

## Non-goals

- Interactive TUI.
- Configuration wizard.

## Required files or modules

- `packages/cli/src/cli.ts`
- `packages/cli/package.json`

## Required types or contracts

- CLI positional commands and flags.

## Required behavior

Offline analysis:

```bash
signalglass analyze <file> [--report terminal|json|html] [--output <file>]
```

Live ingress:

```bash
signalglass ingress --config <file> [--port <port>]
```

Trace management:

```bash
signalglass traces list
signalglass traces show <id>
signalglass traces report <id> [--report html|json]
signalglass traces export <id> --output <file>
```

- The CLI wires packages together and contains no domain logic.
- Help text is accurate and concise.

## Acceptance criteria

- [ ] `signalglass analyze samples/messy-agent-run.json` works.
- [ ] `signalglass ingress --help` documents the command.
- [ ] Unknown commands print usage and exit with non-zero status.
